# 🧮 Scientific Calculator Pro

A modern desktop **Scientific Calculator** developed using **Java Swing**. The application provides both basic and scientific mathematical operations with a clean dark-themed user interface, calculation history, and memory functions.

---

## 📌 Project Description

Scientific Calculator Pro is a Java desktop application that performs basic and advanced mathematical calculations. It features a custom-built expression parser that evaluates expressions following the correct order of operations (BODMAS/PEMDAS), along with scientific functions such as trigonometric operations, logarithms, powers, square roots, and factorials.
The project is designed using **Java Swing** with a modern user interface inspired by the Windows Scientific Calculator.

---

## ✨ Features

### Basic Operations
- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- Decimal calculations
- Parentheses support

### Scientific Functions
- √ Square Root
- x² Square
- ^ (xʸ) Power
- n! Factorial
- sin(x)
- cos(x)
- tan(x)
- log(x)
- ln(x)
- π Constant

### Memory Functions
- MC (Memory Clear)
- MR (Memory Recall)
- MS (Memory Store)
- M+ (Memory Add)
- M− (Memory Subtract)

### Additional Features
- Expression History
- Backspace
- Clear (C)
- Dark Theme UI
- Rounded Buttons
- Custom Expression Parser
- Error Handling

## 🚀 How to Run
### Prerequisites

- Java JDK 17 or above
- IntelliJ IDEA (or any Java IDE)
- Maven

### Steps

1. Clone the repository:
```bash
git clone https://github.com/its-rashmi0508/ScientificCalculator.git
```

2. Open the project in IntelliJ IDEA.

3. Allow Maven to download dependencies.

4. Run:

```
Main.java
```

5. The Scientific Calculator will launch.
